Prsentation des donnes utiliser

Les fichier de donnes sont mis disposition sont contenu dans le dossier data. Ce repertoire contient 4 couches dinformation gographiques :

  • LA_ARRONDISSEMENT_S.shp, les limites des 122 arrondissements du Sngal. Base de donnes gospatiales prioritaires du Sngal lchelle 1 / 1 000 000. https://www.geosenegal.gouv.sn/, 2014.
  • LA_DEPARTEMENT_S.shp, les limites des 42 dpartements du Sngal. Base de donnes gospatiales prioritaires du Sngal lchelle 1 / 1 000 000. https://www.geosenegal.gouv.sn/, 2014.
  • LA_REGION_S.shp, les limites des 14 rgions du Sngal. Base de donnes gospatiales prioritaires du Sngal lchelle 1 / 1 000 000. https://www.geosenegal.gouv.sn/, 2014.
  • LA_FRONTIERE_INTERNATIONALE_FRONTIERE_ETAT_L, les frontires nationnales du Sngal. Base de donnes gospatiales prioritaires du Sngal lchelle 1 / 1 000 000. https://www.geosenegal.gouv.sn/, 2014.

Pour importer ces fichiers shapefiles, utilisez la fonction st_read du package sf

library(sf)
region <- st_read("data/LA_REGION_S.shp")
